# 🔍 Empathy Map

**Phase:** Discover · **Emoji:** 🔍

> Verbatim Design Action from the PRCSS* template. The emoji reflects the primary phase this action belongs to.

### Description

---

An empathy map is a canvas divided into four quadrants (say, think, do, feel) placed around the user. Completing the map can give you an overview of who the user is and identify discrepancies in perceptions of the same user by different team members (thus intervening to resolve conflicts).
