# 🎯 Personas

**Phase:** Define · **Emoji:** 🎯

> Verbatim Design Method from the PRCSS* template. The emoji reflects the primary phase this method belongs to.

### Description

---

Each persona is a reference model that represents a particular type of user. Technically, we call them behavioral archetypes when they focus on capturing a range of behaviors (e.g., "conscious decision makers") without representing defined personalities or sociodemographics. The more realistic the archetype (name, age, household composition, etc.), the more it becomes a true persona that perfectly expresses the needs, desires, habits, and cultural context of a particular user group.
