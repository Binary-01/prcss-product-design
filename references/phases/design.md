# 🧪 Design

**Color:** purple · **Emoji:** 🧪

## Intent
Turn the selected direction(s) into tangible design artifacts. This is where information architecture, interaction design, and visual design converge.

## Typical methods to add to this phase
- User flows / task flows
- Information architecture & sitemap
- Wireframes (low → mid fidelity)
- Prototypes (clickable)
- Hi-fi UI screens
- Design system contributions (tokens, components, patterns)
- UX writing / microcopy passes
- Accessibility pass (WCAG 2.1 AA)

## Typical outputs
- A prototype ready to test.
- Final screens + specs.
- Components and tokens contributed back to the design system.

## Exit criteria
A user can complete the core task(s) in a prototype — or a stakeholder can see the final experience — without needing the designer to narrate. Ready for Test.
