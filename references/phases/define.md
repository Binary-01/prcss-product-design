# 🎯 Define

**Color:** red · **Emoji:** 🎯

## Intent
Frame the problem. This is the convergent phase of the first diamond — you are narrowing down Discover's findings into a crisp problem statement and a prioritized scope.

## Typical methods to add to this phase
- Affinity mapping of research notes
- Jobs-to-be-Done statements
- Problem statement / How-Might-We reframe
- Personas or proto-personas
- User journey / service blueprint (current state)
- Prioritization matrix (impact × effort / RICE / MoSCoW)
- Scope / non-scope statement

## Typical outputs
- One sharp problem statement.
- A prioritized list of user needs / opportunities.
- Explicit success criteria (ties back to Objectives and Metrics).
- A shared understanding of scope and constraints.

## Exit criteria
The team can recite the problem and the top 1–3 opportunities without looking at the doc. If stakeholders still disagree about *what* you are solving, stay in Define.
