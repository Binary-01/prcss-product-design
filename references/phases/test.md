# 🪄 Test

**Color:** orange · **Emoji:** 🪄

## Intent
Validate the design with real users and stakeholders. This is the convergent phase of the second diamond — you're deciding whether to ship, iterate, or loop back to Define / Ideate.

## Typical methods to add to this phase
- Moderated usability testing (5-user studies)
- Unmoderated remote testing
- A/B or multivariate tests
- Concept testing with target users
- Stakeholder review / design critique
- Accessibility audit with assistive tech users

## Typical outputs
- A prioritized list of issues with severity.
- Validated (or invalidated) design hypotheses.
- A revised prototype or a Go/No-Go recommendation.

## Exit criteria
Top-severity issues are fixed (or explicitly accepted) and the core task success rate meets the threshold you set in Define.
